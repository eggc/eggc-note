
- ルータでつないだパソコンや通信機器はローカルエリアネットワーク(LAN)に参加してることになる。
- LANに参加しているデバイスはIPアドレスを与えられる
- LANに参加しているデバイス間は通信できる
  - コマンド arp -a とかするとLANに参加しているデバイスから他のデバイスのMACアドレスとIPアドレスが見える（全部じゃないようだ）
  - windows や mac でファイルブラウザでネットワークのところを見ると見えたりする（ファイル共有設定が必要）
  - お互いのIPアドレスが分かってれば ping xxx.xxx.xxx.xxx のような感じで疎通確認できる
    - デバイスのファイアウォールに阻まれることもある
    - ルータのファイアウォールに阻まれることもある
- LAN がインターネットに接続している場合はインターネット上のデバイスとも通信できる
  - LAN上のデバイスからインターネットのデバイスにリクエストするタイプの通信は自由にできる
  - インターネット上のデバイスからLANのデバイスにリクエストするタイプの通信は普通はルータで遮断している
